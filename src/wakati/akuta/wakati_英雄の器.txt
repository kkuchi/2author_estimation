。 その 顔 が また 、 どれ も いつ に なく 微笑 を 浮べて いる の は 、 西楚 の 覇王 の 首 を あげた 今日 の 勝 戦 の 喜び が 、 まだ 消え ず に いる から であろう 。 ―― 「 そう か ね 。 」 鼻 の 高い 、 眼光 の 鋭い 顔 が 一 つ 、 これ は やや 皮肉な 微笑 を 唇 頭 に 漂わ せ ながら 、 じっと 呂馬通 の 眉 の 間 を 見
